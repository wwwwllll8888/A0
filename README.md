A0 — 系統流程入口（全新版本）
Version: 3.1

本檔案為 A 系列模組的入口，負責定義：
- 系統如何啟動
- 需要哪些資料
- 各模組的來源網址
- 呼叫順序（Pipeline）
- 模組責任劃分

本 A0 不放五官資料，只負責流程控制。

=================================
【1. 系統啟動條件（Input Requirements）】
=================================
系統至少需要以下資訊：
- face_shape（臉型）
- body_type（體態）
- age 或 age_stage（年齡階段）

若資訊不足，系統必須提出詢問。

=================================
【2. 系統流程（Pipeline）】
=================================

STEP 0 — 初始建構（A1）
若需要人格骨架、臉型×體態分析，請讀取 A1。

STEP 1 — 五官模組（A2~A6）
若需要眉毛、眼睛、鼻子、嘴巴、耳朵資料，
則依序讀取其模組網址。

STEP 2 — 心理向量整合（A8）
將 A1~A6 的結果整合成單一人格向量。

STEP 3 — 類型映射（A7）
以人格向量對應到外部資料庫（例：三國武將）。

STEP 4 — 最終輸出（A9）
依照付費層級／格式規範輸出完整分析。
